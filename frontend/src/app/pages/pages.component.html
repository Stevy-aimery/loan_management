Bienvenue dans l'application de Gestion des Prêts

Cette application vous permet de gérer les prêts d'appareils aux utilisateurs.

Utilisez le formulaire pour enregistrer les informations d'un prêt d'appareil.

Scanner un code QR

Scannez le code QR d'un appareil pour être redirigé vers le formulaire prérempli avec les informations de l'appareil.

Nouveau prêt

Créez un nouveau prêt en remplissant le formulaire avec les informations de l'utilisateur et de l'appareil.
